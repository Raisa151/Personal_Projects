Problem:
Parkinson's Diseases Classification using Machine Learning.

Dataset:
The dataset is collected from kaggle.
Link : https://www.kaggle.com/datasets/vikasukani/parkinsons-disease-data-set

It has 195 voice recordings from 31 people, 23 with Parkinson's disease (PD). The dataset has 22 atributes. The details can be found in the 
giiven link.

Label:
0 -> healthy
1 -> PD

Classifiers:

*   Support Vector Machine
*   Logistic Regression
*   Decision Tree Classifier
*   Random Forest Classifier (Information gain)
*   Random Forest (Entropy)
*   KNN
*   XG_boost

Evaluation Metrics:

*   Accuracy
*   f1-score
*   Confusion Matrix
