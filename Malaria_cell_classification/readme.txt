Detail: 
This is a image classification problem. Malaria cells are catagorized into two types, which are parasitized and uninfected. 
A CNN model is built to classify these two catagories. 

Dtaset:
Dataset is collected from national library of medicine. 
Link: https://lhncbc.nlm.nih.gov/LHC-publications/publications.html

It is also available in kaggle.
Link: https://www.kaggle.com/datasets/iarunava/cell-images-for-detecting-malaria

The dataset is too big to upload here.

Objective:
To learn, 1. OpenCV, 2. Pillow, 3. Building CNN model.
